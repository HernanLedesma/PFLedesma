# Phaser Music

Este proyecto consiste en una página web que simula una tienda virtual y de asesoramiento sobre instrumentos musicales, utilizando tecnologías como HTML, CSS, Bootstrap, AOS y SASS.

## Tecnologías utilizadas
- HTML
- CSS
- Bootstrap
- AOS
- SASS
